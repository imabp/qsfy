## What it does?

```
{
	"a": "b"
}
```
URL Friendly Format-> `https://someexample.com/object?a=b`


```
{
	"a": {
		"b" : "c"
	}
}
```
URL Friendly Format-> `https://example.com/object?a[b]=c`



```
{
	"a": {
		"b" :  {
				"c" : "d"
			}
		}
}
```
URL Friendly Format-> `https://example.com/object?a[b][c]=d`
